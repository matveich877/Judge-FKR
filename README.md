## Как обновить реестр судей

Реестр подтягивает данные из CSV-файла автоматически при загрузке страницы.

### Шаг 1. Подготовь CSV

Формат файла `data/judges.csv`:

```csv
ФИО,Регион,КУМИТЭ,КАТА,Категория,Дата_присвоения,Дата_продления,Статус,Приказ_присвоения,Приказ_продления
Иванов Иван Иванович,Москва,да,нет,С1К,15.03.2023,15.03.2025,Всё в норме,https://disk.yandex.ru/i/xxx,https://disk.yandex.ru/i/yyy
```

**Колонки:**
- `ФИО` — полное имя судьи
- `Регион` — город / область
- `КУМИТЭ` — `да` или `нет`
- `КАТА` — `да` или `нет`
- `Категория` — `С1К`, `С2К`, `С3К`, `ЮС` или `ССВК`
- `Дата_присвоения` — формат `ДД.ММ.ГГГГ`
- `Дата_продления` — формат `ДД.ММ.ГГГГ` (можно оставить пустым)
- `Статус` — `Всё в норме`, `Требуется продление (менее 90 дней)`, `Необходимо продление`, `Нет данных`
- `Приказ_присвоения` — ссылка на документ (необязательно)
- `Приказ_продления` — ссылка на документ (необязательно)

> **Важно:** Сохраняй CSV в кодировке **UTF-8** с разделителем **запятая**.

### Шаг 2. Замени файл в репозитории

1. Открой репозиторий на GitHub
2. Перейди в папку `data/`
3. Нажми **Add file → Upload files**
4. Загрузи новый `judges.csv` (замени старый)
5. Нажми **Commit changes**

Изменения появятся на сайте в течение **1–2 минут**.
